# OceanHealthVisualizationApp

# Purpose 
This app is meant to be practice for using Shiny and R to visualize data. 

# Inspiration
This project is inspired by my previous Power BI report which also visualized Ocean health data in a dashboard. Although, this time I generated fake data myself insteaf of sourcing data from Kaggle.

# Data Meaning
The data represents corresponding coral bleaching percentage and Ocean pH for some bodies of water (mainly oceans and seas). Each body of water has a pH and bleaching percentage attached to it

# Features
For this app, I created a UI that visualizes ocean data on different charts and allows filtering of the visible data. It includes a navbar and stylized visuals using CSS. 

# Future improvements
In the future, I could find a a bigger data set to use so that I can paint a bigger picture. To do this, I would also need to add a mechanism to assign colors to bodies of water. Another improvement would be to add more charts, especially for comparison so that the data can be better contexualized and thus be more usefull (if it were real data).
